## #dictionary key-value pair

```py
# dictionary key-value pair

student = {
    "id": 1,
    "name": "Mary",
    "age": 22,
    "movies": ["Spider Man", "Thor", "The Godfather"]
}
```

```py

student['name']

student['movies']
result: ['Spider Man', 'Thor', 'The Godfather']

student['movies'][0]
result: Spider Man
```

```py
class Dog:
    name = "Binnie"
    age = 2
    color = "Brown"
    breed = "Golden Retriever"

    # function (Dog method)
    def bark(self):
        print("Woof! Woof!")
    def sitting(self):
        print("I am sitting now")
    def hungry(self, food_name):
        print(f"I am hungry, I need {food_name}!")

my_dog = Dog()

my_dog.bark()
result: Woof! Woof!

my_dog.sitting()
result: I am sitting now

my_dog.hungry("Pizza")
result: I am hungry, I need Pizza!
```
